````markdown
## 一、总体目标

建立一套可解释、可回测、可每日运行的“首板涨停回调”选股策略。

策略核心逻辑：

> 股票首次涨停吸引资金关注后，在随后几个交易日缩量回调或高位横盘；如果涨停日开盘价支撑仍然有效、行业仍强、尾盘出现承接，则尾盘买入，博弈次日反包或短期修复。

第一阶段不使用机器学习，先通过明确规则找到真正有效的规律。

最终分工：

- 掘金量化：提供行情数据、历史回测数据和分钟数据。
- AuroraAI：保存数据、运行选股、生成评分、展示候选榜单及回测结果。
````

| 指标    | 量化条件                 | 得分 |
| ----- | -------------------- | -: |

|       | 前20日涨幅10%～20%        |  2 |
|       | 前20日涨幅>20%           |  0 |
| 成交量放大 | T0成交量÷前5日均量为1.5～3.0  |  4 |
|       | 1.2～1.5或3.0～4.0      |  2 |
|       | <1.2或>4.0            |  0 |
| 成交额   | ≥5亿元                 |  3 |
|       | 2亿～5亿元               |  2 |
| 换手率   | 5%～15%               |  3 |
|       | 3%～5%或15%～25%        |  2 |
|       | <3%或>25%             |  0 |

| 行业强度  | 申万一级行业评分≥70          |  2 |

说明：

* 1.5～3倍放量优先；

* 缩量涨停暂时不作为最高质量首板；
* 未来根据回测结果调整分段，但不得直接根据某一阶段最高收益过度拟合。

## 六、首板实体和关键价位

定义：

```text
O0 = T0涨停日开盘价
C0 = T0涨停收盘价
R  = C0 - O0
```

首板实体位置比例：

```text
position_ratio = (回调最低价 - O0) / (C0 - O0)
```

关键价格：

```text
1/3位置 = O0 + 0.33 × R
1/2位置 = O0 + 0.50 × R
```

`O0`同时作为：

* 买入前的回调支撑位；
* 买入后的结构止损价。

---

## 七、三类买点模型

三个模型一起开发、一起回测，但结果必须分开统计。

### A1：首板实体1/2回踩

目标区间：

```text
position_ratio = 0.40～0.60
```

特征：

* 回调较浅；
* 趋势相对较强；
* 更容易维持首板后的强势结构。

### A2：首板实体1/3回踩

目标区间：

```text
position_ratio = 0.23～0.43
```

特征：

* 回调更深；
* 更接近涨停资金的低位成本区；
* 潜在反弹空间较大，但走弱风险也更高。

### B：涨停价附近横盘

建议硬条件：

1. T+1～T+4没有任何一天收盘相对C0回撤超过5%；
2. 最近2～3日价格区间振幅不超过6%；
3. 最新成交量不超过T0成交量的70%；
4. 至少连续两日没有明显放量；
5. 最新收盘价不低于 `C0 × 0.96`；
6. 行业评分不低于70。

特征：

* 没有明显回踩；
* 涨停附近承接较强；
* 可能爆发更快；
* 也需要防止高位出货。

### 模型归类规则

为了减少重叠：

* 回调最低位置在0.40～0.60：A1；
* 回调最低位置进入0.23～0.43：A2；
* 低于0.23但未跌破O0：深回调观察组，V1暂不交易；
* 有效跌破O0：淘汰；
* 一直保持在C0附近且满足横盘条件：B。

A1和A2区间存在少量交叉时，按距离0.50或0.33哪个中心点更近进行唯一归类。

## 八、健康回调条件

主要研究 `T+2～T+5` 买点。

### 硬条件

1. 当前价格低于或接近T0收盘价；
2. 进入A1、A2或B模型之一；
3. 买入前没有收盘跌破O0；
4. 买入日成交量不超过T0成交量的70%；
5. 回调期间没有跌停；
6. 回调期间没有再次连续涨停；
7. 没有明显放量下跌；
8. 行业评分不低于70；
9. 回调日收盘位于日内振幅上半区。

收盘位置计算：

```text
close_location = (close - low) / (high - low)
```

基础要求：

```text
close_location >= 0.50
```

高质量标准：

```text
close_location >= 0.65
```

### 开盘价保护版本

回测两个版本：

* P1严格版：回调期间最低价始终不低于O0；
* P2刺破观察版：盘中最多跌破O0约1%，但当日收盘重新站上O0。

V1实际候选优先使用P1，P2作为对照组。

### 直接淘汰

出现以下任一情况直接淘汰：

* 回调日收盘跌破O0；
* 成交量超过T0成交量且股价下跌；
* 出现跌停；
* 出现重大利空或异常停牌；
* 行业评分跌破50；
* 股票进入风险警示状态。

## 十一、尾盘买入

计划买入时间：

```text
14:40～14:55
```

分钟级确认：

1. 当前价格仍高于O0；
2. 尾盘没有突然放量下杀；
3. 价格从日内低点出现企稳或回升；
4. 行业指数没有明显跳水；
5. 大盘风险状态不是极弱；
6. 买入时策略评分仍达到75分。

在只有日线的长期回测阶段，使用收盘价近似买入价，但必须标记：


## 十二、止损规则

### V1主止损：S1盘中严格止损

```text
止损价 = T0涨停日开盘价 O0
```

买入后只要盘中价格首次低于O0，即触发止损。

执行细则：

1. 正常盘中跌破O0：

   * 不假设精准成交在O0；
   * 回测按下一根1分钟K线开盘价成交。
2. 次日跳空低开在O0以下：

   * 按实际开盘价退出；
   * 不能按O0模拟成交。
3. 止损后即使重新站回O0：

   * 当日不重新买入。
4. 每笔交易记录止损空间：

```text
stop_distance = (buy_price - O0) / buy_price
```

如果买入价距离O0过近，S1可能容易被正常波动触发，因此需要单独分组统计。

### 对照止损

虽然S1是V1主方案，回测仍保留：

* S0：无价格止损，仅时间退出；
* S1：盘中首次跌破O0，主方案；
* S2：跌破 `O0 × 0.99`；
* S3：收盘跌破O0；
* S4：跌破O0后连续15分钟未收回。

重点验证S1：

* 是否显著降低大额亏损；
* 被止损后重新上涨的比例；
* 被止损后又达到止盈位的比例；
* 最大回撤是否改善；
* 总收益是否因误杀而明显下降。

## 十三、止盈与退出矩阵

止盈和持有时间暂不主观确定，通过回测选择。

| 方案 | 止盈      | 止损/风险退出  | 最晚退出       |
| -- | ------- | -------- | ---------- |
| E1 | 不设      | S1或无止损对照 | 次日开盘       |
| E2 | +2%或+3% | S1       | 次日10:00    |
| E3 | +3%     | S1       | 次日收盘       |
| E4 | +3%或+5% | S1       | 最多持有3日     |
| E5 | +3%或+5% | ATR止损对照  | 最多持有3日     |
| E6 | 不设      | 不设       | 分别持有1、2、3日 |

日线回测阶段主要统计：

* 次日开盘收益；
* 次日最高潜在收益；
* 次日最低波动；
* 次日收盘收益；
* 持有2日、3日收益；
* MFE；
* MAE。

注意：

> 日线同时出现最高价达到止盈、最低价跌破止损时，无法判断先后顺序，不能直接判定交易盈利。必须交给分钟线复核。
